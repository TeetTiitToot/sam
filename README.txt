1) Installer phantomjs : http://phantomjs.org/download.html
2) modifier le fichier sam.config : login, mot de passe, path vers l'exec de phantomjs
3) lancer le jar : 
	java -jar sky-0.0.1-SNAPSHOT-jar-with-dependencies.jar
	
Le fichier de config des logs est dans le jar, log4j.properties.

NB: les quetes ne sont pas toutes dedans (notamment celles des invasions qui tournent!)
